[ 피그마 컴포넌트 ]
-> 만들고자 하는 요소나 그룹을 선택후 상단의 컴포넌트 버튼을 누르거나, ctrl+alt+K 버튼으로 생성레이어에서 이동도 가능함

1. 일반 그룹화와는 다른 재사용성을 위한 공용 모듈화 요소
2. 원본 개념이 있어서 원본을 수정하면 카피 버전역시 자동으로 수정됨
3. 원본은 채워진 다이아몬드, 카피버전은 비워진 다이아몬드
4. 일반적으로 컴포넌트는 프레임 바깥쪽에 만들고 관리하는 것이 좋음
5. 제작된 컴포넌트는 Assets 메뉴에 있음
    -> 또는 상단 메뉴의 Resources 메뉴에도 있음
6. 컴포넌트 내부에 컴포넌트를 묶어서 작업할 경우
    - 내부 컴포넌트 -> Variant(변형) - 단어의 뜻을 볼 때 기본형으로부터 변경시에 사용됨
