​‌‌‍⁡⁢⁣⁣[ 라우터 컴포넌트 호출시 데이터 전달하기 ]⁡​

- 일반적인 웹페이지에서는 GET/POST 방식으로 페이지간 데이터를 전달함
-> 리액트 라우터를 사용할 때 데이터 전달은 어떻게 하나?

1. Link 라우터 컴포넌트 사용시
<Link to="라우터주소" state={{속성명: 값}} />

받는 방법)
-> 보내는 페이지
useLocation 라우터 객체를 사용한다

코드)
-> 받는 페이지
import { useLocation } from 'react-router-dom'

const loc = useLocation();
const sel = loc.state.속성명

-> useLocation 생성변수 하위의 state 객체 하위 보낸 속성명으로 읽어옴

2. useNavigate 사용하여 전달값 보내기

코드)
import { useNavigate } from 'react-router-dom'

const goNav = useNavigate();
goNav(보낼 라우터 주소, {state: 값})

-> 받는 방법은 위의 Link보내기 시 받는 방법과 동일
-> useLocation으로 받음
