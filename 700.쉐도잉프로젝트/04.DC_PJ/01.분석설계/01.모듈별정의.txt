[2] 메뉴 버튼 모듈
    모듈명 : MenBtn
    (1) 구조정의
        Root> img_box + title_box + button_box
    (2) 객체화
        {
            "img_src":"이미지 경로",
            "tit":"작은타이틀^큰타이틀",
            "btn":"버튼 문구"
        }
        -> "tit" 속성은 ^로 구분한다
        -> 컴포넌트별 객체화 데이터 반영은 기본
    (3) 기능정의
        1) 버튼 오버시 하이라이트
                (mouseover, mouseout) + css
        2) 버튼 클릭시 페이지이동
            (click)  + location(라우트 이동)

[3] 비디오 소개 모듈
    모듈명 : VidIntro
    (1) 구조정의
        Root > video box(>video+img) + title box(>heading+ p tag)
    (2) 객체화
        {
            "vsrc": 유튜브 동영상 경로,
            "stit": 작은타이틀,
            "btit": 큰타이틀,
            "sum": 요약소개,
            "decs":  설명,
            "link": 관련링크
        }
    (3) 기능정의
        1) 동영상영역 클릭시 동영상 재생/멈춤
        2) 링크 새창 열기
